# 单人测试隐私声明
## ISEF项目：帕金森辅助设备

### 项目性质说明
本项目为**单人测试**项目，仅涉及研究者本人作为测试者，不涉及其他人类参与者。

### ISEF合规状态
根据ISEF指南，单人测试项目通常**不需要**完整的人体受试者审查，但仍需遵循基本的数据保护原则。

### 简化的隐私保护措施

#### 1. 数据收集范围
- **测试者**：研究者本人
- **数据类型**：个人传感器数据（手指弯曲、EMG、IMU）
- **用途**：算法开发和系统测试
- **性质**：工程研究，非医疗用途

#### 2. 数据管理
- **存储**：本地个人设备存储
- **标识**：使用个人标识符（如"SELF_TEST"）
- **保留**：根据研究需要保留
- **删除**：项目完成后可选择删除

#### 3. 风险评估
| 风险类型 | 风险等级 | 说明 |
|----------|----------|------|
| 隐私风险 | 极低 | 仅涉及个人数据 |
| 安全风险 | 低 | 本地存储，个人控制 |
| 合规风险 | 极低 | 单人测试，简化要求 |

#### 4. ISEF报告建议表述
```
本项目采用单人测试模式，研究者本人作为唯一测试者进行系统验证。
数据收集仅涉及个人传感器数据，用于算法开发和性能评估。
项目遵循ISEF工程研究指南，不涉及其他人类参与者。
```

### 注意事项

#### 1. 如果未来扩展项目
如果计划邀请其他参与者测试，需要：
- 实施完整的隐私保护框架
- 获得知情同意
- 进行IRB审查（如学校要求）

#### 2. 数据共享考虑
- 如果计划在ISEF展示数据，确保：
  - 不包含敏感个人信息
  - 使用匿名化数据
  - 说明数据来源为研究者本人

#### 3. 项目连续性
- 保留测试数据和日志
- 记录测试过程和结果
- 为未来扩展做好准备
